# VibeEdit

**An AI-native video editor for macOS — now drivable entirely by voice.**

VibeEdit is a Swift-native video editor (SwiftUI + AppKit, AVFoundation) where you and an AI agent edit on the same timeline. This hackathon build adds a **hands-free accessibility layer**: you can play, navigate, and edit video without a mouse or keyboard.

> Requires macOS 26 (Tahoe) on Apple Silicon.

## Build & run

```bash
swift build
swift run
```

On first launch, grant **Microphone** and **Speech Recognition** permissions to use voice mode. Agent mode also needs an ElevenLabs key/agent ID (Settings → Voice).
